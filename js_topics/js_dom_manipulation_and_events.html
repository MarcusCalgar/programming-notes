DOM Manipulation and Events

DOM - Document Object Model

  • A tree-like (tree of "nodes") representation of the contents of the page.
  • Targeting nodes with selectors is possible with CSS selectors.
  • There are also relationship selectors that are properties of the node. E.g.: node.firstElementChild or node.lastElementSibling.
  • HTML elements are turned into nodes, and nodes have a lot of built in methods and properties attached to them.
    • Query Selectors: E.g.: element.querySelector(selector): returns the first match of the selector.
      If the selector has multiple matches, the elements are returned in a nodelist. This is NOT an array, even if it looks, and sometimes behaves like an array. Some array methods are missing from nodelists, but it is possible to convert nodelists to Arrays using Array.from() or the spread operator.
    • Element creation: creates a new element, e.g.: const div = document.createElement('div'); Note, that this just creates the element, but doesn't add it to the DOM.
    • Append element: parentNode.appendChild(childNode);
    • Remove element: parentNode.removeChild(child);. Returns a reference to the child.
    • When accessing a CSS style from JS, use camelCase. E.g.: div.style.background-color doesn't work, but div.style.backgroundColor and div.style['background-color'] does.
    • Editing Attributes:
      • div.setAttribute('attributeToSet', 'value');
      • div.getAttribute('attributeToGet');
      • div.removeAttribute('attributeToRemove');
    • Working with Classes
      • div.classList.add('newClass');: adds a new class to the element.
      • div.classList.remove('classToRemove');: removes a class from the element.
      • div.classList.toggle('class');: if the element doesn't have the class, it add is, else, it removes it.
    • Adding text content: div.textContent = 'Hello World!';
    • Adding HTML content: div.innerHTML = '<span>Hello World</span>';
  • To prevent JS running before the DOM finished loading, add the script tag at the end of the HTML. Or when linking from an outside source, use: <script src="script.js" defer></script>
