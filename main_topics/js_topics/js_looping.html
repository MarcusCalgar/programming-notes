Looping Through Arrays

  • for...of loop: if we have a collection, you can iterate through it: for (const item of collection){}.
  • map(): do something to a collection and return a new collection. E.g.: const newArray = oldArray.map(doSomethingFunction);
  • filter(): test each item against a rule and return a new array with the items that pass the rule. E.g.: const newArray = oldArray.filter(someBooleanTest);
  • map() and filter() are usually used with function expressions. E.g.: const filteredArray = unfilteredArray.filter((item) => item.someBooleanTest());
  • The break statement immediately exits a loop.
  • The continue statement jumps to the next iteration of the loop.
  • while(someBooleanCondition){doTheThing}: it runs the code until the condition is satisfied.
  • The do...while loop works similarly to the while loop but guarantees at least one iteration.
  • Ternary operators cannot have break and continue, it will throw a syntax error.
  • for loops can be labelled, then the labels can be used to break out of multiple levels deep iterations. E.g.: outer: for(..;..;..){for(..;..;..){break outer}} breaks out of the outer loop.
