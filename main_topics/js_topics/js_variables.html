Variables

  • A variable is like a 'box' for data.
  • Use console.log(); to log messages to the console.
  • Use alert("message"); to make the website a popup with the content of "message".
  • Declare a variable with the let keyword.
    • It is possible to declare variables on a single line, but it isn't recommended. E.g.: let user="John", age=25, message="hello";
  • The keywords var and let work very similarly, however there is difference in scope.
  • The content of a variable can be easily changed. E.g.: let name="John"; name="Roger";.
    Note: Some functional programming languages like Scala and Erlang forbid the value change of declared variables.

Variable naming

  • A variable name must only contain digits, letters and the symbols $ and _ , however, the first character cannot be a digit.
  • Longer variable names should be in camelCase.

Constants

  • Variables that do not change are declared by the const keyword. Such variables cannot be reassigned.
  • Constants that can be hard coded usually written in uppercase. E.g.: const COLOR_RED="#F00";
  • Creating a new variable for a new thing isn't a problem. A couple more lines of code won't affect performance too much.
