Arrays

  • Arrays are special variables that hold multiple values.
  • It's common practice to declare arrays with const.
  • Elements can be accessed by their index number. Arrays are zero indexed.
  • JavaScript arrays can hold different types of data at the same time. E.g.: strings, numbers, functions, other arrays, objects, etc...
  • In JavaScript arrays always use numbered indexes, they cannot have names as in hashes.
  • Arrays are Objects, but Objects use named indexes, while arrays use numbered indexes.

Properties and methods

  • array.length: returns the number of elements
  • array.sort(): sorts the array
  • array.push("new item"): puts the "new item" to the end of the array. It also returns the new length value.
  • Array.isArray("variable"): checks whether "variable" is an array or not.
  • array instanceof Array;: returns true if the checked variable is an array.
  • array.toString();: returns a string with the comma separated values in the array.
  • array.join(" * "): like toString() but the separator can be specified.
  • array.pop(): removes (and returns) the last item from the array.
  • array.shift(): removes (and returns) the first element of the array, while shifts all remaining elements down one index.
  • array.unshift(): adds a new element to the beginning of the array, while renumbering the indexes. Also returns the new length.
  • delete array[n]: removes the element from index n and leaves a "hole" which is undefined.
  • array1.concat(array2, array3): merges two arrays. Returns a new array and leaves the others intact.
  • array1.concat("Peter"): concat can also merge strings.
  • array.splice(2, 0, "Item1", "Item2");: adds items to an array. The first number defines the position where the new items need to be, the second is number of items to be deleted. The rest is the items that are getting added. It returns an array with the deleted items.
  • array.slice(1, 4): slices out part of an existing array from the first index number, up to (but not included) the second number, to a new array. It leaves the original array intact.
