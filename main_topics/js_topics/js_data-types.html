Data Types

JavaScript is dynamically typed which means that data types exist, but variables are not bound to them.

  1. Numbers
  2. BigInt: for integer numbers larger than 9007199254740991 (253-1) where precision matters. Add n to the end of the number to create.
  3. String
  4. Boolean (logical type): true and false.
  5. null: In JS, this is not a null-pointer, or a reference to a non-existing object. It rather means "nothing", "empty", or "value unknown".
  6. undefined: "value not assigned".
  7. Symbols: closely tied to Objects.
  8. Objects
