To be successful at problem solving, follow these steps:

  1. Understand the problem: the first step to be able to solve any problem is to define what the problem actually is.
    • Write it down in plain text
    • Draw some diagrams if necessary
  2. Plan: break down the large problem into manageable smaller ones. Some useful questions are:
    • Does it have an interface?
  3. Write pseudocode
